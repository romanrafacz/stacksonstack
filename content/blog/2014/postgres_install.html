Let's install postgresql on CentOS 64bit.

First thing we need to do is install the repo so we can use the yum pagage manager.
We're going to install postgresql94, go ahead and issue the following commands. < 


rpm -Uvh http://yum.postgresql.org/9.4/redhat/rhel-6-x86_64/pgdg-centos94-9.4-1.noarch.rpm



sudo yum install postgresql94 postgresql94-jdbc postgresql94-plpython postgresql94-devel

Once you install postgres, you need to init the database. If you just try to start the database, you'll get the error message below. 


service postgresql-9.4 start

/var/lib/pgsql/9.4/data is missing. Use "service postgresql-9.4 initdb" to initialize the cluster first.
                                                           [FAILED]

Switch to root and issueservice postgresql-9.4 initdb, you can then start posgresql-9.4 . Then swith to postgres and issues the psql command. You can then create a user for your database.


create user pgadmin with password '******';



alter user pgadmin superuser;
